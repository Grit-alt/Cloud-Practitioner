##### &nbsp;**S3 Static Website Hosting Tutorial**



***Step 1: S3 Bucket Creation***

\- Logged into the AWS Management Console

\- Created an S3 bucket named `freshlyground-cafe-website`

\- Selected the appropriate AWS region

\- Disabled “Block all public access” to allow website access



***Step 2: Website File Upload***

\- Uploaded all website files including HTML, CSS, and images

\- Verified that `index.html` was included as the main entry file



***Step 3: Static Website Hosting Configuration***

\- Enabled static website hosting in the S3 bucket properties

\- Set `index.html` as the index document



 ***Step 4: Bucket Policy Configuration***

\- Applied a bucket policy to allow public read access to website files



&nbsp;***Step 5: Website Access and Testing***

\- Accessed the website using the S3 static website endpoint URL

\- Confirmed that the website loads correctly in the browser
