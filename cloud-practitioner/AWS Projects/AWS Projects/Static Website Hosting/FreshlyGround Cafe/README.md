*** Static Website Hosting for FreshlyGround Cafe (AWS S3)***



This project was completed as part of the AWS re/Start Portfolio Project.

It demonstrates how a static website for a café can be hosted using Amazon S3.



The website includes the café name, menu items, business description, and contact information.

Amazon S3 static website hosting was used as a cost-effective and scalable solution to improve the café’s online presence.
